# Лабораторная работа №3. Фильтрация изображений и морфологические операции, вариант 10

## Реализация BMP класса

Класс `BMPImage` загружает 24-битные BMP файлы, хранит пиксели в непрерывном массиве `std::unique_ptr<std::byte[]>` и учитывает выравнивание строк до 4 байт. Поддерживаются заголовки произвольного размера (стандартные 40 байт и расширенные).

## Выполненные операции

1. **Ранговый фильтр** — маска «косой крест» из 5 пикселей (координаты: (-1,-1), (-1,1), (0,0), (1,-1), (1,1)). Для каждого пикселя значения в маске сортируются, выбирается элемент с рангом 4 (четвёртый из пяти).

2. **Разностное изображение** — для монохромных изображений вычисляется XOR, для полутоновых — модуль разности. При необходимости выполняется контрастирование (умножение на коэффицент с ограничением 255).
