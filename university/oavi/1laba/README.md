# Лабораторная работа №1. Цветовые модели и передискретизация изображений, вариант 10

## Реализация BMP класса

Класс `BMPImage` загружает 24-битные BMP файлы, хранит пиксели в непрерывном массиве `std::unique_ptr<std::byte[]>` и учитывает выравнивание строк до 4 байт. Поддерживаются заголовки произвольного размера (стандартные 40 байт и расширенные).

## Выполненные операции

1. **Цветовые модели** — выделение каналов R, G, B, преобразование RGB->HSI с сохранением яркостной компоненты, инвертирование интенсивности.
2. **Передискретизация** — растяжение билинейной интерполяцией, сжатие усреднением блока, двухпроходная (растяжение + сжатие) и однопроходная передискретизация.

## Демонстрация

Для демонстрации выполненной работой предлагается выполнить `make build-and-run`.
