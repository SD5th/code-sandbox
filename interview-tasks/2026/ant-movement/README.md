## Базовое решение задачи с муравьём на C++
### Вводные
Доска с муравьём представлена в виде статического массива статических массивов bool\[2001\]\[2001\]. Размер выбран с учётом того, что муравей не сможет перейти за столбец/строку 1999 и 999, так как сумма цифр будет больше 25.

Из этих же соображений ни одна функция дальше не проверяет выход за границы массива, так как известно, что муравей не сможет попасть в крайнюю клетку.

### Алгоритм решения
1. Доска заполняется значениями *false*.
2. Из клетки (1000, 1000) начинает работать рекурсивная функция, считающая количество посещённых клеток.

### Описание рекурсивной функции:
1. Начинаем с клетки (x, y):
   1) Если сумма цифр x и y больше 25, возвращаем 0.
   2) Если matrix\[x\]\[y\] уже *true* (клетка посещена), возвращаем 0.
2. Помечаем клетку (x, y) посещённой, matrix\[x\]\[y\] = *true*.
3. Присваем локальному счётчику значение 1.
4. Увеличиваем счётчик на значения, полученные из вызова функции для прилегающих клеток.
5. Возвращаем значения счётчика.

Никаких оптимизаций не предполагается, так как поле достаточно мало (меньше 1000x1000), программа отрабатывает моментально. Такой алгоритм называется *обходом графа в глубину*, он же **DFS**. 

### Сборка и запуск
Решение собирается с помощью CMake на C++ 17. CMake вызывается внутри builder Docker-контейнера на Alpine Linux, используются статические библиотеки. На следующую стадию переходит только исполняемый файл ant-movement.

### Использование
В файле Makefile предоставлены скрипты для работы с обычным контейнером и debug-контейнером. Названия говорящие. Доступные команды:
Команды:
  - make docker-build
  - make docker-run
  - make docker-build-and-run
  - make docker-build-debug
  - make docker-run-debug
