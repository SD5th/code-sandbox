# Лабораторная работа №4. Выделение контуров на изображении, вариант 10

## Реализация BMP класса

Класс `BMPImage` загружает 24-битные BMP файлы, хранит пиксели в непрерывном массиве `std::unique_ptr<std::byte[]>` и учитывает выравнивание строк до 4 байт. Поддерживаются заголовки произвольного размера (стандартные 40 байт и расширенные).

## Выполненные операции

1. **Преобразование в полутоновое** — цветное изображение приводится к оттенкам серого по формуле среднего арифметического.

2. **Оператор Прюитт 5×5**:
   - `G = |Gx| + |Gy|` 
   - `Gx` — горизонтальные градиенты (маска с верхней и нижней строками)
   - `Gy` — вертикальные градиенты (маска с левым и правым столбцами)

3. **Нормализация** — все градиентные матрицы приводятся к диапазону 0–255.

4. **Бинаризация** — для матрицы G подбирается порог, значения выше порога становятся белыми (контуры), ниже — чёрными (фон). Также сохраняется инвертированная версия.

## Демонстрация

Для демонстрации выполненной работой предлагается выполнить `make build-and-run`.
