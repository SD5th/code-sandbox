# Лабораторная работа №2. Обесцвечивание и бинаризация растровых изображений, вариант 10

## Реализация BMP класса

Класс `BMPImage` загружает 24-битные BMP файлы, хранит пиксели в непрерывном массиве `std::unique_ptr<std::byte[]>` и учитывает выравнивание строк до 4 байт. Поддерживаются заголовки произвольного размера (стандартные 40 байт и расширенные).

## Выполненные операции

1. **Полноцветное → полутоновое** — яркость пикселя вычисляется как среднее арифметическое с сохранением в 24-битный BMP (R=G=B=яркость).
2. **Полутоновое → монохромное** — адаптивная бинаризация методом Вульфа с локальным окном 15×15 и параметром k = 0.5. Для разных типов изображений подбираются оптимальные параметры.

## Демонстрация

Для демонстрации выполненной работой предлагается выполнить `make build-and-run`. Для разных изображений опытным путём подобрал аргументы, при которых получается наилучший результат.
